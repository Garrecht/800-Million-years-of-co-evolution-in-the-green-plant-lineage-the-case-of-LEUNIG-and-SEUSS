# 800-Million-years-of-co-evolution-in-the-green-plant-lineage-the-case-of-LEUNIG-and-SEUSS
Supplemental data for the scientific paper "800 million years of co-evolution in the green plant lineage – the case of LEUNIG and SEUSS transcriptional co-regulators" by Garrecht, J.; Rössner, C.; Sreechakram, V. N. S.; Li, Q.; Braun, S; Golz; J. F.; Becker; A.

[STATUS: In submission]

---

**Script 1: Extract alpha helices.py:**  
  Python script for extraction of α-helix positions from AlphaFold protein models via PyMol (Schrödinger, 2010; Abramson et al., 2024).

**Script 2: Extract beta sheets.py:**  
  Python script for extraction of β-sheet positions from AlphaFold protein models via PyMol (Schrödinger, 2010; Abramson et al., 2024).  

**Script 3: Residue visualization.py:**  
  Python script for visualizing interactions of the LUFS-domain of LUG with SEU in AlphaFold protein models via PyMol (Schrödinger, 2010; Abramson et al., 2024).  

**Script 4: Interacting residues.py:**  
  Python script for extraction of interacting residues from AlphaFold protein models via PyMol (Schrödinger, 2010; Abramson et al., 2024). 

**Script 5: Interacting residues.R:**  
  R script for visualizing positions of extracted interacting residues from script 4.  

**Script 6: elm_number_bars.R:**  
  R script for visualizing the total number of individual SliMs identified by elm.eu.org in a subset of SEU/SLK-like sequences (Kumar et al., 2023).

**Script 7: elm_position.R:**  
  R script for visualizing of the position of all SliMs identified by elm.eu.org in a subset of SEU/SLK-like sequences (Kumar et al., 2023).  

**Script 8: Y2H plate cropping.ijm:**  
  Fiji (ImageJ) macro for semi-automatic cropping of Y2H assay plate pictures (Schindelin et al., 2012).  

**Script 9: Y2H analysis.R:**  
  R script using the gitter package to measure yeast colony sizes and to determine presence/absence of protein interactions in a Y2H assay (Wahih and Parts, 2021).  

**Script 10: BiFC merging and cropping.ijm:**  
  Fiji (ImageJ) macro to split and merge color channels for BiFC analysis (Schindelin et al., 2012).

**Script 11: CorrelationPlots.R:**  
  R script for the creation of correlation plots of gene expression data.

**Script 11: Complex visualization.R**
  R script for the visualization of AlphaFold protein models via PyMol (Schrödinger, 2010; Abramson et al., 2024).
